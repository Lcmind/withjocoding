# FPS Reaction Time Test 🎯

무료 FPS 반응속도 측정 게임! 타겟을 빠르게 클릭하고 내 반응속도를 측정하세요!

## ✨ 주요 기능

- 🎯 **타겟 클릭 게임** - 랜덤 위치에 나타나는 빨간 원 클릭
- ⏱️ **정확한 반응속도 측정** - ms 단위로 측정
- 🏆 **3가지 난이도** - 쉬움/보통/어려움
- 📊 **내 기록 TOP 10** - 로컬 저장, 서버 불필요
- 🥇 **등급 평가** - 프로게이머급부터 평균까지
- 📱 **SNS 공유** - Twitter, Facebook, Instagram, 링크 복사
- 🌍 **10개 언어 지원** - 한국어, 영어, 일본어, 중국어, 스페인어, 프랑스어, 독일어, 포르투갈어, 러시아어, 아랍어
- 💰 **AdSense 수익화** - 상단/하단/사이드 광고 배치
- 🔍 **SEO 최적화** - 메타태그, OG태그, 구조화 데이터
- 📱 **모바일 완벽 지원**
- 🎨 **Tailwind CSS** - 깔끔하고 모던한 디자인
- 🔒 **완전 프라이버시 보호** - 사진 서버 전송 없음

## 🎮 게임 방식

1. 난이도 선택 (타겟 크기)
2. 게임 시작 버튼 클릭
3. 3-2-1 카운트다운
4. 빨간 원이 랜덤 위치에 등장
5. 최대한 빠르게 클릭!
6. 10개 완료 후 결과 확인

## 📊 등급 시스템

| 평균 반응속도 | 등급 | 순위 |
|---|---|---|
| < 150ms | 🏆 프로게이머 급! | 상위 1% |
| 150-200ms | ⚡ 매우 빠름! | 상위 5% |
| 200-250ms | 🎯 빠름 | 상위 15% |
| 250-300ms | 👍 평균 이상 | 상위 30% |
| 300-400ms | 😊 평균 | - |
| 400ms+ | 🐌 느림 | 연습 필요 |
