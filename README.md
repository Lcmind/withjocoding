# 전생 직업 분석 AI 🔮

사진 한 장으로 AI가 당신의 전생 직업을 15가지 중에서 진단하는 바이럴 웹 애플리케이션입니다!

## ✨ 주요 기능

- 🤖 **온디바이스 AI 분석**: Transformer.js + CLIP 모델 사용
- 🔮 **15가지 전생 직업**: 왕/여왕, 기사, 마법사, 음유시인 등
- 🎲 **SSR 히든 결과**: 9% 확률로 특별한 결과 (신/시간여행자/기록말소)
- 📱 **SNS 공유 기능**: 카카오톡, 인스타그램, 트위터, 페이스북
- 🔒 **완벽한 프라이버시**: 사진이 서버로 전송되지 않음
- 💯 **완전 무료**: 온디바이스 AI로 서버 비용 제로
- 🎯 **바이럴 최적화**: 인스타 스토리용 이미지 다운로드

## 🔮 전생 직업 카테고리

### 👑 왕족/귀족
- 왕/여왕 (King/Queen) - 당당하고 카리스마 있는 리더
- 귀족 (Noble) - 우아하고 세련된 상류층
- 영주 (Lord) - 든든하고 책임감 있는 통치자

### ⚔️ 전사/무인
- 기사 (Knight) - 정의롭고 용감한 전사
- 궁수 (Archer) - 집중력과 냉철함의 화신
- 검투사 (Gladiator) - 불굴의 투지를 가진 영웅

### 🔮 지성/신비
- 마법사 (Wizard) - 신비로운 힘을 다루는 현자
- 현자/학자 (Sage) - 지혜롭고 온화한 스승
- 점성술사 (Astrologer) - 운명을 읽는 신비로운 존재

### 🎭 예술/자유
- 음유시인 (Bard) - 자유로운 영혼의 예술가
- 궁중화가 (Court Painter) - 아름다움을 포착하는 예술가
- 모험가 (Adventurer) - 미지를 탐험하는 도전자

### 🕵️ 특수
- 스파이 (Spy) - 그림자 속의 첩보원
- 대상인 (Merchant) - 협상과 교역의 달인
- 성직자 (Priest) - 영혼을 치유하는 성인

### ✨ SSR 히든 결과
- 신 (God) - 1% 확률 (극찬)
- 시간여행자 (Time Traveler) - 3% 확률 (엽기)
- 기록말소 (Forgotten) - 5% 확률 (극딜)

## 🐛 문제 해결

### AI 모델이 로드되지 않아요
- 인터넷 연결 확인
- 브라우저 콘솔에서 에러 메시지 확인
- 다른 브라우저에서 시도

### 카메라가 작동하지 않아요
- 브라우저 카메라 권한 확인
- HTTPS 환경에서만 작동 (localhost는 OK)
- 다른 브라우저에서 시도

### 분석이 부정확해요
- 얼굴이 잘 나온 사진 사용
- 명확한 얼굴 표정이 보이는 사진 사용
- AI는 재미 목적이므로 정확도 100% 아님

### 이미지 다운로드가 안돼요
- 브라우저 다운로드 권한 확인
- 팝업 차단 해제

## 📝 개발 노트

### 왜 온디바이스 AI를 선택했나요?

1. **프라이버시**: 사진이 서버로 전송되지 않음
2. **무료**: API 비용 없음
3. **글로벌**: 제한 없이 전세계 사용 가능
4. **빠름**: 서버 통신 불필요

### 왜 Transformer.js + CLIP을 선택했나요?

- **CLIP**: OpenAI의 강력한 이미지-텍스트 매칭 모델
- **사전 학습**: 별도 학습 데이터 불필요
- **Zero-shot**: 새로운 키워드로 즉시 분류 가능
- **얼굴 분석**: 표정과 특징을 텍스트로 매칭
- **경량**: 브라우저에서 실행 가능

### 바이럴 포인트는?

1. **감동적인 스토리**: 전생 이야기가 감성 자극
2. **SSR 히든 결과**: 가챠 요소로 재도전 유도
3. **SNS 공유**: 스토리용 이미지 다운로드
4. **비교 욕구**: 친구들과 비교하고 싶음
5. **전생 테스트 트렌드**: 글로벌하게 통하는 주제
